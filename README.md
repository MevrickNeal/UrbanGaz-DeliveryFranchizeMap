# UrbanGaz-DeliveryFranchizeMap
URBN GAZ Tactical Dashboard

A high-end, automotive-inspired logistics dashboard for URBN GAZ. This application manages LPG delivery routes across Dhaka with a premium, glassmorphic UI.

🚀 Features

Live Logistics Map: Integrated Leaflet.js with real-time Dhaka coordinates.

Delivery Navigation: Animated "Road Path" transitions between sectors (Jolshiri, Khilkhet, Uttara).

Admin Portal: Dedicated interface for project site registration and GPS coordinate management.

Billing Integration: Excel import simulation for franchise billing data.

Premium Design: Whitish glassmorphism aesthetic with custom LPG cylinder iconography.

🛠️ Tech Stack

Framework: React.js

Styling: Tailwind CSS

Animations: Framer Motion

Icons: Lucide React

Maps: Leaflet.js

📦 Installation & Setup

Clone the repository:

git clone [https://github.com/your-username/urbn-gaz-dashboard.git](https://github.com/your-username/urbn-gaz-dashboard.git)
cd urbn-gaz-dashboard


Install dependencies:

npm install


Run the development server:

npm run dev


Build for production:

npm run build
